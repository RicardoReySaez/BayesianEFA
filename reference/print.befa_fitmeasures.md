# Print Bayesian Fit Measures

Displays a formatted table of Bayesian fit indices from a
`befa_fitmeasures` object.

## Usage

``` r
# S3 method for class 'befa_fitmeasures'
print(x, digits = 3, ...)
```

## Arguments

- x:

  A `befa_fitmeasures` object from
  [`befa_fit_measures()`](https://ricardoreysaez.github.io/BayesianEFA/reference/befa_fit_measures.md).

- digits:

  Integer. Number of decimal places to display.

- ...:

  Ignored.
